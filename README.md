# My Portfolio Website

A personal portfolio website built from scratch using vanilla HTML, CSS, and JavaScript.

## Live Demo

[View Portfolio]([https://RihazRamzaan.github.io/portfolio](https://rihazramzaan.github.io/Portfolio/))

## Built With

- HTML5
- CSS3
- Vanilla JavaScript

## Features

- Responsive design
- Hero entrance animations
- Typing effect
- Scroll reveal animations
- Nav frost effect on scroll
- Active nav link highlighting

## Sections

- Hero — introduction and photo
- Skills — languages, frameworks, databases and tools
- Projects — featured work with links
- Contact — email and social links

## Setup

1. Clone the repository
```bash
git clone https://github.com/RihazRamzaan/portfolio.git
```
2. Open `index.html` in your browser
3. Or use Live Server in VS Code
